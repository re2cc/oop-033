# Practica 3

## Reglas
1. Rango de Humedad (`setHumidityThreshold`):
    - El valor debe estar entre 0 y 100.
    - La humedad se mide como porcentaje, un valor fuera de este rango no tiene sentido.

2. Límite de Temperatura (`setTemperatureThreshold`):
    - El valor debe ser mayor a -273.15° C.
    - Evita errores de entrada de datos que son físicamente imposibles..
